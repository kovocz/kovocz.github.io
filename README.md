## O projektu
Tento projekt vznikl jako platforma pro testování moderních webových technologií a efektivní správu kódu. Hlavním cílem je vytvořit čisté, udržitelné a snadno škálovatelné řešení, které může sloužit jako inspirace nebo základ pro další vývoj. Projekt je neustále ve fázi vylepšování, takže jakákoliv zpětná vazba nebo příspěvek jsou vítány!
